    Interface CertificateInfo

    Cert projection used inside the signing pipeline.

    Distinct from the public X509CertificateInfo type (in types/crypto.ts) which is the v1 public surface — this internal shape carries the precomputed hash + signing-specific aliases that the legacy pipeline produced. Phase 3 will collapse the two shapes once the invoice builders are ported.
